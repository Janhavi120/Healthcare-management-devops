# Healthcare-management-devops
Flask + SQLite based healthcare system with Docker and Terraform.

# Features
Dashboard
Patients
Doctors
Appointments
Medical Records

# Tech Stack
Python Flask
SQLite
Docker
Terraform
Run Locally
python app.py

# Run with Docker
docker-compose up 

Open browser:
http://localhost:9000

# Terraform
cd terraform terraform init terraform apply

# Access Hosted Application
Open browser:
https://healthcare-management-devops-4.onrender.com
